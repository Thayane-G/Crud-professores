# CRUD de Professores

Este projeto é um sistema simples de CRUD (Create, Read, Update, Delete) desenvolvido em PHP com integração ao MySQL. O objetivo do projeto foi praticar conceitos básicos de desenvolvimento web backend.

---

## 📌 Funcionalidades

- Cadastro de professores
- Listagem de professores
- Visualização de dados individuais
- Edição de informações
- Exclusão de registros

---

## 💻 Tecnologias utilizadas

- PHP (linguagem principal)
- MySQL (banco de dados relacional)
- HTML
- CSS (Bootstrap)
- SQL

---

## ⚙️ Como executar o projeto

1. Clone ou baixe o repositório
2. Coloque os arquivos em um servidor local (como XAMPP)
3. Crie o banco de dados no MySQL
4. Configure a conexão no arquivo `conexao.php`
5. Inicie o servidor local
6. Acesse o projeto pelo navegador utilizando o endereço local do servidor (localhost)

---

Durante o desenvolvimento deste projeto, pratiquei:

- Conexão com banco de dados em PHP utilizando MySQL
- Implementação de operações CRUD

---

## 📁 Estrutura do projeto

- `index.php` → listagem dos professores
- `professor-create.php` → cadastro de professores
- `professor-edit.php` → edição de professores
- `professor-view.php` → visualização de dados
- `acoes.php` → lógica do CRUD
- `conexao.php` → conexão com banco de dados
- `mensagem.php` → exibição de mensagens
- `navbar.php` → componente de navegação
